4 - DEMONSTRAÇÃO DE CÓDIGO

O laborátorio

PRIMEIRO CÓDIGO:

Nome: MARK

Função: Serve para destacar um pedaço de texto, criando um efeito visual idêntico ao de um marca-texto amarelo.

Exemplo: Esse texto tem uma palavra destacada aqui.

Resultado: As palavras chaves serão as primeiras a serem vistas

SEGUNDO CÓDIGO

Nome: STRONG

Função: Serve para indicar que um trecho de texto tem grande importância ou urgência. Visualmente, o navegador deixa o texto em negrito.

Exemplo: O evento começará pontualmente às 09:00h.

Resultado: A palavra vai se destacar entre as outras

TERCEIRO CÓDIGO

Nome: DEL

Função: Serve para indicar que um texto foi deletado, removido ou atualizado. Visualmente, o navegador exibe o texto com uma linha riscada no meio.

Exemplo: O preço do ingresso promocional era R$ 150,00, mas agora é R$ 90,00!

Resultado: Indicar que um texto já não é mais assim, que foi "deletado"
